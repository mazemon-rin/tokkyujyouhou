# X Rules

Purpose:
ブログへのアクセス誘導

Recommended Length:
50〜120文字

Structure:

1. 話題性
2. 商品名
3. 気になるポイント
4. 記事URL

Example:

AirPods 4が話題👀

セール価格になっているため在庫切れ前に確認しておきたい商品です。

詳しくはこちら
[URL]

## Avoid

- 長文
- ハッシュタグ3個以上
- 過度な煽り
- 根拠のない最安値表現

## Recommended Hashtags

#Amazon
#お得情報

## Posting Times

- 07:00
- 12:00
- 20:00
